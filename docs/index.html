Bloomreach Forge Marketplace

What is the Marketplace?

The Bloomreach Forge Marketplace is an Essentials plugin that brings addon discovery directly into your brXM development environment. Browse, search, and get installation instructions for Forge addons without leaving the CMS.

Key features:

  • Browse Addons - Search and filter available Forge addons
  • Multi-Source Support - Aggregate addons from Forge, partners, and internal sources
  • Version Compatibility - See which addon versions work with your brXM version
  • Installation Guide - View step-by-step installation instructions

For Addon Developers

Publishing your addon to the Marketplace is simple:

  1. Add a .forge/addon-config.yaml to your repository
  2. Set up the GitHub workflow for auto-generation
  3. Release your addon - the descriptor is generated automatically

See Creating Descriptors for the complete guide.

For Platform Operators

Add custom addon sources to include partner or internal addons:

  • Create a manifest file hosting your addon descriptors
  • Register the source via REST API or JCR configuration
  • Addons appear alongside Forge addons in the Marketplace

See Custom Sources for details.
