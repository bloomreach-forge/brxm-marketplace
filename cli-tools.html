CLI Tools

Overview

Two CLI tools are available from GitHub Releases:

  • descriptor-generator.jar - For addon developers
  • manifest-generator.jar - For manifest maintainers

descriptor-generator

Generates forge-addon.yaml for addon repositories by combining a config file with pom.xml metadata. 


# Download
curl -sLO https://github.com/bloomreach-forge/brxm-marketplace/releases/latest/download/descriptor-generator.jar

# Usage
java -jar descriptor-generator.jar [options]

Options:
  -c, --config <file>    Config file path (default: .forge/addon-config.yaml)
  -o, --output <file>    Output file path (default: forge-addon.yaml)
  -p, --pom <file>       POM file path (default: pom.xml)
  -v, --verbose          Enable verbose output
  -h, --help             Show help

Example


java -jar descriptor-generator.jar \
  --config .forge/addon-config.yaml \
  --output forge-addon.yaml \
  --verbose

Artifacts Configuration

By default, a single artifact is generated with target: parent and scope: compile. For multi-artifact add-ons, configure explicitly in your config: 


# .forge/addon-config.yaml
artifacts:
  - target: cms
    description: "CMS module"
  - target: site/components
    artifactId: my-addon-site
    description: "Site components"

Valid targets: parent, cms, site/components, site/webapp, platform

Valid scopes: compile (default), provided, runtime, test

manifest-generator

Generates addons-index.json by scanning a GitHub organization for repositories containing forge-addon.yaml files. 


# Download
curl -sLO https://github.com/bloomreach-forge/brxm-marketplace/releases/latest/download/manifest-generator.jar

# Usage
java -jar manifest-generator.jar [options]

Options:
  -g, --org <name>       GitHub organization to scan (required)
  -o, --output <file>    Output file path (default: addons-index.json)
  -t, --token <token>    GitHub token for API access
  -v, --verbose          Enable verbose output
  -h, --help             Show help

Example


java -jar manifest-generator.jar \
  --org bloomreach-forge \
  --output addons-index.json \
  --token $GITHUB_TOKEN \
  --verbose

This tool is primarily used by the Marketplace's GitHub Actions workflow to generate the official Forge addon index.

Downloading Specific Versions

To download a specific version of the tools: 


# Latest
curl -sLO https://github.com/bloomreach-forge/brxm-marketplace/releases/latest/download/descriptor-generator.jar

# Specific version
curl -sLO https://github.com/bloomreach-forge/brxm-marketplace/releases/download/1.0.0/descriptor-generator.jar
